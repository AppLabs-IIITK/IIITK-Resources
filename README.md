## ğŸ“– Contributing

Want to add resources? Follow these steps:

2. **Add** your files in the appropriate folder
3. **Create** a Pull Request with a clear description

### File Naming Convention

- Use descriptive names: `DS_MidSem_2024.pdf`
- Include subject code if available: `CS201_EndSem_2023.pdf`
- For slides: `Week01_Introduction.pptx`

### Supported Formats

| Type | Extensions |
|------|------------|
| Documents | `.pdf` |
| Presentations | `.ppt`, `.pptx` |
| Notes | `.md`, `.txt` |
| Code | `.py`, `.java`, `.cpp`, etc. |
